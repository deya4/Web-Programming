How to Search and Enroll for Classes at Binary Hub Learning

This guide will show you how to look up and enroll in classes offered by Binary Hub Learning.

Steps to Register:

  1. Go to the enrollment portal button located on the top right of the page.
  2. Enter your username and password and click Log In.
  3. On the left, select "Class Search and Enroll."
  4. Select the term (e.g., Winter Quarter).
  5. At the top of the page, enter a keyword to search for specific classes, then click the Search button.
  6. On the left-hand side, filters will display for you to narrow your search results:
    • Subject
    • Class Status
    • Number of Credits
    • Location
    • Instruction Mode
    • Class Meeting Days (Days of the Week)
    • Class Start Times
    • Class End Times
  7. Select a course and click the class hyperlink to view details.
  8. The class information window will display.
  9. Click on the close (x) button to close the class information window.
  10. To enroll in this class, click on the Enroll button (you may add classes to a "cart," and you'll have the option to bulk enroll in these classes).
  11. Select the Next button at the top right of the page.
  12. The "Review Class Information" will be displayed. Input waitlist preference if the class is full, and select the Accept button.
  13. The "Review and Submit" page will display. Once you click the Submit button, a submit confirmation will display.
  14. Select the Yes option to confirm submission.
